(static) setSetting(setting)

プラグイン設定を行う

compositionプラグインには以下の設定項目があります。

設定項目 型 説明 デフォルト値
off boolean プラグイン無効設定。無効にする場合はtrueを指定。 false
container DOM|jQuery|string メッセージ表示先となるコンテナ要素。 なし
wrapper string メッセージを出力する要素のタグ名またはタグ生成文字列。'li'や、'<span class="error-msg">'のような指定ができ、指定された文字列から生成した要素が各メッセージ要素になります。 なし(テキストノードとして表示)
message string バリデートエラー時に表示するメッセージ。メッセージの記述形式はh5.ui.validation.MessageOutput.setMessageのmessageプロパティと同じ形式です。 デフォルトルール毎にデフォルトのメッセージが用意されており、それらが使用されます。
formatter function バリデートエラー時に表示するメッセージフォーマッタ。フォーマッタの記述形式はh5.ui.validation.MessageOutput.setMessageのformatterプロパティと同じ形式です。 なし
label string バリデーション対象のプロパティに対応するラベル名 バリデーション対象のプロパティ名

各設定項目について、共通設定とプロパティ毎の設定を記述できます。以下、記述例です。 


{
	container: 'ul.composition',
	wrapper: 'li',
	property: { // 各プロパティ固有の設定
		userid: { // プルパティ名
			label: 'ユーザ名',
			message: '{label}は必須です'
		}
	}
}
