(static) setSetting(setting)

プラグイン設定を行う

styleプラグインには以下の設定項目があります。

設定項目 型 説明 デフォルト値
off boolean プラグイン無効設定。無効にする場合はtrueを指定。 false
errorClassName string バリデートエラー時に適用するクラス名 なし
successClassName string バリデート成功時に適用するクラス名 なし
validatingClassName string 非同期バリデートの結果待機時に適用するクラス名 なし
replaceElement DOM|jQuery|string|function クラス適用対象要素をDOM,jQuery,セレクタの何れかで指定。関数を設定した場合は第1引数にデフォルトは各プロパティのフォーム部品要素が渡され、その関数が返す要素が対象要素になります。 各プロパティのフォーム部品要素

各設定項目について、共通設定とプロパティ毎の設定を記述できます。以下、記述例です。 


{
	errorClassName: 'error', // バリデートエラー時に適用するクラス名
	successClassName: 'success', // バリデート成功時に適用するクラス名
	validatingClassName: 'validating', // 非同期バリデート待ちの場合に適用するクラス名
	property: { // 各プロパティ固有の設定
		userid: { // プルパティ名
			errorClassName: 'id-error',
			replaceElement: function(element) {
				// クラス設定対象要素は親要素にする
				return $(element).parent();
			}
		},
		name: {
			off: true
		// nameプロパティについてプラグインによるクラス設定を無効
		}
	}
}
Parameters:
Name Type Description
