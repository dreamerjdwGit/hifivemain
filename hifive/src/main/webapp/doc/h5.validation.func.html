(static) size(value, min, max) → {boolean}

値のサイズが範囲内であるかどうかを判定し、判定結果をtrueまたはfalseで返します

サイズの判定対象となる値は配列、文字列、プレーンオブジェクトの何れかです。 配列の場合は配列の長さ、文字列の場合は文字数、プレーンオブジェクトの場合はプロパティをサイズとして扱い、判定の対象となります。

その他の型の値の場合はfalseを返します。

第2引数でサイズの下限値、第3引数でサイズの上限値を指定し、その範囲内のサイズであるかどうかを判定します。上限値、下限値ともに境界を含めます。

値がnullまたはundefinedの場合はtrueを返します。

Parameters:
Name Type Description
value Any 判定する値
min integer 下限値
max integer 上限値
